# Tech With Tim — Browser Games

Standalone browser games. No install, no build step — just open the HTML file.

## Games

### Tic Tac Toe (`tictactoe.html`)
Classic Tic Tac Toe vs an unbeatable minimax AI.
- Click any cell to place your X
- Background chiptune music + sound effects
- Score tracker across games

### Dead Zone (`shooter.html`)
Retro top-down shooter with wave-based progression.
- **Move:** Arrow keys or WASD
- **Aim:** Mouse
- **Shoot:** Left click
- **Pause:** ESC
- 3 enemy types (Walkers, Runners, Tanks) with increasing difficulty
- Pixel-art sprites drawn entirely in Canvas
- Chiptune music + SFX via Web Audio API

## Running
Open any `.html` file directly in Chrome, Firefox, or Edge. No server needed.
